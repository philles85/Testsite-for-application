About this project

This project was created to be able to get to know eachother. Through this application the user can become friends with other employes!

Important information

To access this API and framework, user has to be connected to the internet

Access API

  • To access this API, users connected to internet or wifi.
  • Use the correct endpoints listed below for proper functionality.

API Endpoints

/allUsers

  • GET: Returns all the users from the external companys API, this is returned in JSON format
  • GET Status Codes:
  • 200 Success - The list of users is presented
  • 500 Internal Server Error - Server connection issue

/addToFriendsList

  • POST: Sends the userId to server for adding a user as friend
  • Adds the user as a friend to JSON data
  • Returns a status message
  • POST Status Codes:
  • 200 Success - User is added successfully
  • 409 Conflict - User already exists as friend
  • 500 Internal Server Error - Server connection issue

/getFriendList

  • GET: Returns the friends array list for the user in json format
  • GET Status Codes:
  • 200 Success - The friends list was successfully fetched
  • 500 Internal Server Error - Server connection issue

/removeFriend

  • DELETE: Deletes the requested userId in the friends JSON database
  • Users can select a user to delete
  • DELETE Status Codes:
  • 202 Accepted - Request was a success and user is deleted
  • 409 Conflict - Requested user is not in users friends list
  • 500 Internal Server Error - Server connection issue
